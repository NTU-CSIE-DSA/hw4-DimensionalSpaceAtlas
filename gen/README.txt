Options:
  -N            N (default 1e6)
  -T            T (default 1e6)
  -max_v        The maximum possible value of elements in sequence (default 1e9)
  -min_v        The minimum possible value of elements in sequence (default 0)
  -w?           ?: 1, 2, 3
                The weight of the possibility performing operation ? (default 1)
                Too large weight will slow the speed of generators
  -max_len      The maximum possible length of the sequence for all time (default 1e7)
                Should be at least N
  -min_len      The minimum possible length of the sequence for all time (default 0)
  -final_len    The exact length of the sequence after all operations
                -1 means not specified (default -1)
